# Special Parts Request - Quick Guide

## What is it?
A feature that lets students request components **not available** in the current inventory.

## How to Use

### Step 1: Access Special Requests
- Sign in as student
- Click **"Special Requests"** in the sidebar (✨ Sparkles icon)

### Step 2: Create New Request
Click **"New Special Request"** button

### Step 3: Fill the Form

#### Part Details (Required)
- **Part Name**: e.g., "ESP32-CAM WiFi Module"
- **Description**: Detailed specs, features, requirements (min 10 chars)
- **Quantity**: How many you need (1-100)
- **Estimated Price**: Optional, in ₹

#### Product Link (Optional)
- **Website URL**: Amazon, Robu.in, or any online store link
- Helps admin find and order the exact component

#### Product Images (Optional)
- Upload up to **5 images**
- Drag & drop or click to upload
- PNG, JPG up to 10MB each
- Preview before submitting
- Remove unwanted images

#### Request Details (Required)
- **Project**: Link to existing project or select "Other"
- **Purpose**: Why you need this component (min 10 chars)

### Step 4: Submit
- Review all information
- Click **"Submit Request"**
- Wait for admin approval

### Step 5: Track Status
Go to **"Special Requests"** to see:
- ⏰ **PENDING**: Just submitted
- 👁️ **UNDER_REVIEW**: Admin is reviewing
- ✅ **APPROVED**: Request approved!
- ❌ **REJECTED**: Not approved (see reason)
- 📦 **ORDERED**: Component ordered
- ✅ **RECEIVED**: Component received and available

## Tips for Faster Approval

1. **Be Specific**: Include exact model numbers and specifications
2. **Add Links**: Product URLs help admin find the right component
3. **Upload Images**: Clear product images speed up verification
4. **Explain Purpose**: Detailed purpose helps justify the request
5. **Check Price**: Providing estimated price helps with budgeting

## Example Request

```
Part Name: ESP32-CAM WiFi Bluetooth Module
Description: ESP32-CAM with OV2640 camera, supports WiFi and Bluetooth, 
             4MB PSRAM, microSD card slot, supports image WiFi upload
Quantity: 2
Estimated Price: ₹450 per unit
Website: https://robu.in/product/esp32-cam-wifi-bluetooth-module
Images: [Product photo, Pinout diagram]
Purpose: Building IoT surveillance system for final year project. 
         Need camera module with WiFi capability for real-time streaming.
Project: IoT Home Security System
```

## What Happens Next?

1. **Admin Reviews**: HOD/Admin checks your request
2. **Decision**: Approved or Rejected (with reason)
3. **If Approved**: Component gets ordered
4. **Status Updates**: Track progress in Special Requests page
5. **Notification**: You'll be notified when status changes
6. **Once Received**: Component added to inventory, you can request it normally

## Need Help?

- Check the guidelines in the form
- Contact lab assistant or HOD
- Review previous approved requests for examples
